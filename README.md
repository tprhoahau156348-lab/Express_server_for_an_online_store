# Online Store API

Backend המדמה חנות אונליין קטנה, נכתב ב-Node.js ו-Express.
המערכת מנהלת מוצרים, עגלות קניות של לקוחות ותהליכי תשלום (Checkout) מבוססי קבצי JSON מקומיים.

## הגדרת סביבה (Environment Variables)

הפרויקט דורש קובץ `.env` בתיקייה הראשית כדי לרוץ. 
יש ליצור קובץ כזה.
הקובץ צריך לכלול את המשתנים הבאים:
* `PORT`: הפורט עליו ירוץ השרת (לדוגמה 3000).
* `DB_BASE_PATH`: נתיב לתיקיית מסד הנתונים (לדוגמה `./db`).
* `STARTING_BALANCE`: יתרת הפתיחה ללקוח חדש (לדוגמה 500).

## הפעלת הפרויקט

1. התקנת תלויות: `npm install`
2. הפעלת השרת: `node app.js`.

## נתיבי ה-API (Routes)

### כללי
* `GET /`: הודעת פתיחה קצרה.
* `GET /health`: בדיקת תקינות השרת.

### מוצרים
* `GET /products`: רשימת כל המוצרים הגלויים. תומך בפרמטרים לסינון (`inStock`, `maxPrice`, `search`).

### עגלת קניות (דורש `customerId`)
* `GET /cart`: הצגת עגלת הקניות של הלקוח.
* `POST /cart/items`: הוספת מוצר לעגלה (דורש `productId` ו-`quantity` בגוף הבקשה).
* `DELETE /cart/items/:productId`: הסרת מוצר מהעגלה.

### חשבון והזמנות (דורש `customerId`)
* `GET /account/balance`: צפייה ביתרת הלקוח.
* `POST /orders/checkout`: ביצוע הזמנה על בסיס העגלה הנוכחית.
* `GET /orders`: צפייה בהיסטוריית ההזמנות של הלקוח.
